\subsection{超链接应用}
超链接显示有2种：
\begin{itemize}
\item 链接和显示内容一致：\url{https://www.baidu.com}
\item 链接和显示内容不一致：\href{https://www.baidu.com}{百度链接}
\end{itemize}
